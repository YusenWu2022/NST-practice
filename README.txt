简单介绍:
这是一份轻量级简易版本的风格转换代码，主要根据NST的CVPR论文编写，后期加工时融入了res14的图像物体识别程序，使用tkinter设计
了简单的UI，实现了拉动条选择转换权重和在一次转换后不中止前提下以上一次的结果作为中间结果输入进一步转换。通过手动调整不同层
的权重优化结果。
除此之外，初步尝试了快速风格化，具体在quick中给出。
另外，在examples文件夹中提供了小规模的（256，128）案例结果供参考。
使用pyinstaller生成了exe文件，但是没有配置环境情况下体积过大（>400M) 不过仿佛在win10上的可用性和环境与库条件有关（再次发现python不太适合生
成exe文件qwq).
